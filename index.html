Why Reclipse ?

Debugging reactive Software can be unnecessary complicated. We want to change that. With Reclipse we give you all the tools needed to debug your software in a comfortable, powerful way.

Features

Reactive Tree

You have the possibility to inspect your variables, signals and events interacting with each other presented as nodes in a tree. At a glance you can see the values of your signals and variables as well as the their dependencies and dependents. Manupulate the tree as you wish, drag it around, zoom in or out. Whatever it is that improves your overview over the code, the tree is there to help you with it.

Creating pages manually

If you prefer to not use the automatic generator, push a branch named gh-pages to your repository to create a page manually. In addition to supporting regular HTML content, GitHub Pages support Jekyll, a simple, blog aware static site generator. Jekyll makes it easy to create site-wide headers and footers without having to copy them across every page. It also offers intelligent blog support and other advanced templating features.
